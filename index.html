Google Translate

Dynamically translate between languages between two options: Translation API and AutoML Translation.

Translation API Features

What is it?

Provides an pre-programmed interface that translates strings to any supported language by using Neural Machine Translation.

Languages

Supports more than 100 languages that when combined provides thousands of language pairs.

Language detection

The API can automatically detect the source language in case it wasn't provided.

Simple Integration

Yous just need to send the HTML to the API to get the translated text back. No need to extract some text and then put it back to original place.

Scalable

Each user has a daily quota, if is necessary to increase it you can just make a request.

Pay-Per-Use

Client pays only for what is used based on number of characters of text processed.

AutoML Translation(Beta) Features

What is it?

After uploading some language pairs, AutoML enables you to create custom translation models that return specific queries.

Languages

Supports approximately 50 languages.

REST API

It is accessible via Google REST API.

Integration

Is fully integrated with all other Google Cloud services.

Pricing

Translation API

Free tier: $300 free credit to get started with any GCP product for 12 months.

Text Translation: $10 per million characters (including whitespace characters)

Language detection: $10 per million characters*
*This charge only applies to "detect method" calls. Not when the source language is not specified.

AutoML

Training: $76 per hour

Prediction: 1–500,000 characters is free

Training: 500,001–5,000,000 characters is $80 per million characters
